# Guia de Instalação do Spyfall no Termux

## 1️⃣ Requisitos
- Termux atualizado
- Python 3
- Acesso à internet

## 2️⃣ Clonar o repositório do Spyfall
```bash
git clone git@github.com:alanjack77707-afk/Spyfall-termux.git
cd Spyfall-termux


3️⃣ Instalação
Bash
bash install.sh

4️⃣ Executar Spyfall
Bash
spyfall
